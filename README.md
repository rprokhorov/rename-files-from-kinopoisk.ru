# Kinopoisk file renamer

Скрипт помогает хранить файлы фильмов в аккуратном, едином виде.

Он проходит по файлам в указанной папке, ищет каждый фильм через [неофициальное
API Кинопоиска](https://kinopoiskapiunofficial.tech/) и предлагает варианты
переименования. Вы выбираете подходящий — файл переименовывается по шаблону:

```
Название на русском (Title in English) Год.ext
```

## Установка

```bash
pip install requests
```

## Использование

1. Получите бесплатный API-ключ на https://kinopoiskapiunofficial.tech/ и
   сохраните его в переменной окружения:

   ```bash
   export KINOPOISK_API_KEY="ваш-ключ"
   ```

2. Запустите скрипт, указав папку с фильмами:

   ```bash
   python3 rename.py /путь/к/фильмам              # переименовать на месте
   python3 rename.py /путь/к/фильмам -o /sorted   # переместить в другую папку
   python3 rename.py /путь/к/фильмам --dry-run    # показать, но ничего не менять
   ```

Для каждого файла скрипт показывает до 6 совпадений. Доступные действия:

- `1`–`6` — переименовать по выбранному варианту;
- `n` (или Enter) — пропустить файл;
- `e` — ввести более точный поисковый запрос вручную.

Работает на macOS, Linux и Windows.

## Свой паттерн переименования

Шаблон имени задаётся флагом `-p/--pattern`. Доступные плейсхолдеры:

| Плейсхолдер  | Значение                          |
|--------------|-----------------------------------|
| `{rus}`      | название на русском               |
| `{eng}`      | название на английском/оригинал   |
| `{year}`     | год выпуска                       |
| `{duration}` | длительность                      |
| `{ext}`      | расширение файла (без точки)      |

По умолчанию используется `"{rus} ({eng}) {year}"`. Длительность можно
вернуть, добавив `{duration}` в свой паттерн.

```bash
python3 rename.py /фильмы -p "{rus} {year}"
python3 rename.py /фильмы -p "[{year}] {rus} ({eng})"
```

Группы в скобках `( )`, `[ ]`, `{ }` автоматически удаляются, если внутри
пусто (например, у фильма нет английского названия), — лишних `()` или `[]`
в имени не останется.
